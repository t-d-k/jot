Very Brief Building Instructions

Windows

  • Install "Visual Studio Code"
  • Install the C/C++ extension
  • Install the compiler
    • download from microsoft
    • then select "build tools for visual studio"
  • Open the file jot2xml.sln
  • Build

Linux

  • Install QT Creator
  • Open the file jot2xmlqt/jot2xml.pro in QT Creator
  • Build
